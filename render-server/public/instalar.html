Instalar, Ativar e Iniciar

Voce recebeu um comando por email apos a compra. Cole no terminal e pressione Enter:

O comando ja faz tudo automaticamente: baixa, instala, ativa sua licenca e inicia a Zaya.

git clone https://github.com/Shelbyys/zaya-plus.git && cd zaya-plus && npm install && node activate.js SEU-TOKEN && npm start

Substitua SEU-TOKEN pelo token que recebeu. Exemplo:

git clone https://github.com/Shelbyys/zaya-plus.git && cd zaya-plus && npm install && node activate.js TRIAL-D4E95CCA-CBD0-41A5 && npm start

💡 Dicas
●
A Zaya roda enquanto o terminal estiver aberto. Se fechar o terminal, ela para.
●
Para deixar rodando em segundo plano: cd ~/zaya-plus && npm start &
🔒
Todos os seus dados ficam NO SEU computador, na pasta ~/zaya-plus/
🔄
Para reconfigurar tudo do zero: cd ~/zaya-plus && rm .env .license && cp .env.example .env && npm start
🚧 Problemas comuns
"command not found: node" ▼
Instale o Node.js: https://nodejs.org — clique no botao verde LTS e siga a instalacao.
"command not found: git" ▼
Instale o Git: https://git-scm.com — baixe e instale normalmente.
"EADDRINUSE: address already in use" ▼
A porta 3001 ja esta em uso. Mate o processo com o comando abaixo e rode npm start novamente:
lsof -ti:3001 | xargs kill -9
"Token invalido" ▼
Verifique se copiou o token completo, sem espacos extras no inicio ou no final.
"Token ja ativado em outro computador" ▼
Cada token funciona em apenas 1 computador. Entre em contato com o suporte para resolver.
