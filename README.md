# Zaya Plus

Sua assistente pessoal de IA completa. Chat, voz, WhatsApp, Instagram, Facebook Ads, ligacoes, geracao de imagens e videos.

## Instalacao Rapida

```bash
curl -sL https://raw.githubusercontent.com/Shelbyys/zaya-plus/main/install.sh | bash
```

Ou manualmente:

```bash
git clone https://github.com/Shelbyys/zaya-plus.git
cd zaya-plus
npm install
npm start
```

Acesse `http://localhost:3001` e o Setup Wizard vai te guiar.

## Requisitos

- **Node.js 18+** (obrigatorio)
- **Git** (obrigatorio)
- Google Chrome (opcional)
- FFmpeg (opcional, para videos)

## Funcionalidades

- Chat com IA (ChatGPT, Groq, Claude)
- Voz realista (ElevenLabs)
- WhatsApp Bot automatico
- Ligacoes telefonicas (Twilio)
- Instagram & Facebook management
- Ads Dashboard com metricas completas
- Geracao de imagens (DALL-E, Gemini)
- Geracao de videos (Kling, Veo3)
- Pesquisa web automatica
- Agendamentos e lembretes
- Armazenamento na nuvem (Supabase)
