Instalacao simplificada

Um comando. Tudo pronto.

O novo instalador automatico cuida de tudo: instala Node.js, clona o repositorio, ativa a licenca e inicia o servidor.

Como instalar

curl -sL https://raw.githubusercontent.com/Shelbyys/zaya-plus/main/setup.sh | bash -s SEU-TOKEN
Instala Node.js — via nvm, sem precisar de Homebrew. Funciona em Mac e Linux.
Clona o repositorio — baixa a Zaya Plus em ~/zaya-plus
Instala dependencias — roda npm install automaticamente
Ativa a licenca — usa o token passado no comando
Inicia o servidor — abre em http://localhost:3001

Atalho para reabrir

Depois de instalar, basta digitar no terminal:

zaya

O atalho e criado automaticamente durante a instalacao.
