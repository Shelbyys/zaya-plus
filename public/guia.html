Instalando a Zaya

Abra o terminal e execute os comandos abaixo:

$ git clone https://github.com/Shelbyys/zaya-plus.git
$ cd zaya-plus
$ npm install
$ npm start

Acesse http://localhost:3001 no navegador

Configurando WhatsApp

Conecte a Zaya ao seu WhatsApp em minutos

1Selecione WhatsApp no Setup Wizard
2Escolha: Baileys (gratuito) ou WaSender
3Escaneie o QR Code no celular
4Pronto! A Zaya responde suas mensagens automaticamente

🔒 Seus dados ficam no seu computador, 100% privados

Configurando Redes Sociais

Conecte Instagram e Facebook a Zaya

1Acesse developers.facebook.com
